      Software Tools

      To fulfill our responsibilities we use the following tools as follows...

      Tool Expectation
      Telegram
      Communication      		 Use this tool to...
      • Receive and collaborate on assignments from Stake Auxiliaries
      • Collaborate with all Ward Clerks
      • Be notified of callings that need to be
        • Added
        • Updated
        • Removed
        in LCR
      • Be notified of Melchizedek Priesthood advancement to Elder or High Priest
      • Be notified on Meeting Events
      • Be notified on Meeting Agendas
      Google Calendar
      Track Events      		 Use this as a source of truth for Stake Events
      Google Docs
      Document Hub      		 Use this a source of truth for All Stake Documents
      Zoom
      Virtual Collaboration      		 As approved by Stake Presidency, there will be need to use zoom to broadcast or host meetings. You will be expected to know how to set this up so many could view or others could join meetings like High Council Meetings or Firesides

      Telegram Policy

      To help everyone get to know each other, All Stake Leaders are asks to complete their Telegram profile with a photo of themselves and use the appropriate name so all can recognize each other. https://picofme.io/ is a great tool for profile photos.

      Once sign up is done, feel free to join the following channels.
